# SlerpFlow

Official repository for the ICML 2026 paper:

**SlerpFlow: Spherical Trajectory Correction for Rectified Flow Inversion**

SlerpFlow is a training-free geometric correction method for rectified-flow inversion. It revisits inversion from a hyperspherical latent-space perspective and applies spherical trajectory correction to improve reconstruction fidelity and text-guided image editing while keeping the computational cost comparable to first-order solvers.
